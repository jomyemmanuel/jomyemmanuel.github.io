COLLEGE ID CARD

(Ongoing Project)
A Web app to create id cards for students and faculty. The user inputs details of either the student or the faculty member, as a result of which a pdf form of the card is created. There are options to change the college name, address, font, background design and college logo.
Technologies used include HTML, CSS, JavaScript, JQuery for the frontend, SQLite for the Data Base and Django, ReportLab library for the backend.
